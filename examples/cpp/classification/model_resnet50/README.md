# `model_resnet50` (C++)

Pilot migrated C++ example from `core/examples`, now built as a source-first app example
against an installed SiMa NEAT SDK (`find_package(SimaNeat)`).

## Run

```bash
./build.sh
./build/examples/cpp/classification/model_resnet50/model_resnet50 --model /path/to/resnet_50_mpk.tar.gz
```

The exact output path can vary by generator. Use `find build -name model_resnet50`.
